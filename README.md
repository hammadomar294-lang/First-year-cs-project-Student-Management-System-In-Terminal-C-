# First-year-cs-project-Student-Management-System-C-

A console-based management system built using Object-Oriented Programming.

Features:
- Add / Edit / Delete Students
- Add / Edit / Delete Courses
- Enroll Students
- Modify Grades
- Input Validation
- Multi-level Menu System

Built to practice:
- OOP Design
- Vectors
- Input validation
- System architecture
